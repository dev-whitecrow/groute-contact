# 📊 ACTS29 MBTI 구글 시트 DB 연동 가이드

현재 만들어진 MBTI 테스트(index.html)에서 사용자가 입력한 **'이름'**과 **'검사 결과'**를 구글 드라이브의 스프레드시트에 자동으로 차곡차곡 쌓이게 하는 방법입니다. 

개발 지식 없이도 아래 순서대로만 무조건 따라 하시면 5분 안에 자체 DB 서버가 구축됩니다.

---

## 1단계: 구글 스프레드시트 만들기
1. 본인의 구글 드라이브에 접속하여 **새 구글 스프레드시트**를 생성합니다.
2. 파일 이름을 알아보기 쉽게 적습니다. (예: `ACTS29_MBTI_결과DB`)
3. **첫 번째 줄(A1, B1, C1, D1)**에 표(헤더) 제목을 미리 적어줍니다. 이것은 헷갈리지 않기 위함입니다.
   * `A1`셀: **타임스탬프**
   * `B1`셀: **이름**
   * `C1`셀: **결과 타입** (A, B, C)
   * `D1`셀: **결과 이름**

## 2단계: 구글 앱스 스크립트(GAS) 설정하기
1. 열어둔 스프레드시트 상단 메뉴에서 **[확장 프로그램] ➔ [Apps Script]**를 클릭합니다.
2. 새 창이 열리면 화면 중앙 편집기에 있는 기본 코드(`function myFunction() { ... }`)를 **전부 지웁니다.**
3. 아래 코드를 복사해서 **그대로 붙여넣기** 합니다.

```javascript
function doPost(e) {
  // 현재 연결된 스프레드시트의 1번째 시트(활성 시트)를 가져옴
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  
  // 넘겨받은 파라미터 값 추출
  var name = e.parameter.name;
  var type = e.parameter.type;
  var title = e.parameter.title;
  var time = e.parameter.time;
  
  // 스프레드시트 마지막 줄에 새로운 행 추가 (A, B, C, D열 순)
  var rowData = [
    time,     // 타임스탬프
    name,     // 입력한 이름
    type,     // 알파벳 A~C
    title     // 결과 텍스트
  ];
  sheet.appendRow(rowData);
  
  // 작업 성공 메시지 반환
  return ContentService.createTextOutput("성공적으로 저장되었습니다.");
}
```

4. 상단의 **디스켓 모양(저장)** 버튼을 클릭하거나 `Ctrl+S` (Mac은 `Cmd+S`)를 눌러 저장합니다. (프로젝트 이름은 아무거나 적어도 됩니다)

## 3단계: 나만의 웹 앱 배포(URL 발급)
1. Apps Script 화면 우측 상단에 파란색 **[배포]** 버튼을 클릭하고 **[새 배포]**를 선택합니다.
2. 톱니바퀴 아이콘(유형 선택)을 클릭하고 **웹 앱(Web App)**을 선택합니다.
3. 배포 설정:
   * **설명:** (비워두거나 MBTI DB라고 적으세요)
   * **실행 주체:** `나`를 선택합니다.
   * **액세스 권한이 있는 사용자:** 반드시 **`모든 사용자`**를 선택하세요! (가장 중요합니다. 그래야 일반 유저들이 폼을 제출할 수 있습니다)
4. 권한 승인:
   * 최초 배포 시 "권한 검토" 창이 뜹니다. 본인의 구글 계정을 클릭합니다.
   * 경고 화면("확인되지 않은 앱")이 나올 텐데 겁먹지 마시고 **"고급"** 버튼을 누른 뒤 **"~로 이동(안전하지 않음)"**을 클릭합니다.
   * 허용(Allow) 버튼을 누릅니다.
5. 로딩이 끝나면 길고 복잡한 주소인 **[웹 앱 URL]**이 발급됩니다. 이 주소를 `복사`하세요.

## 4단계: MBTI HTML 코드에 주소 붙여넣기
1. 아까 만들어드린 `index.html` 파일을 코드 편집기(텍스트 편집기, VSCode 등)로 엽니다.
2. 파일의 가장 아래쪽(자바스크립트 스크립트의 제일 첫 줄, 대략 238번 줄 근처)에 보시면 아래와 같은 구역이 있습니다.

```html
    // ==========================================
    // 💡 [ACTS29 DB 연동 설정]
    // 구글 앱스 스크립트(Google Apps Script)에서 배포된 웹 앱 URL을 아래에 붙여넣으세요.
    // 비워둘 경우, DB에는 전송되지 않고 화면 결과만 표시됩니다.
    // ==========================================
    const GOOGLE_SHEET_WEBHOOK_URL = "여기에_URL을_붙여넣으세요"; 
```

3. `"이 사이의 공간에"` 방금 복사한 긴 **[웹 앱 URL]** 을 붙여넣습니다. (따옴표는 지우면 안 됩니다!)
4. 파일을 저장합니다.
