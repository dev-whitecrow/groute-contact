# 🚀 ACTS29 MBTI 테스트 웹 무료 배포 가이드

현재 만들어진 `index.html` 파일 하나만 있으면 정말 1분 만에 전 세계 누구나 접속할 수 있는 **링크(URL)**를 발급받을 수 있습니다.

개발 지식이 전혀 없는 기획자나 마케터도 가장 쉽고 빠르게 무료로 배포할 수 있는 두 가지 방법을 소개합니다.

---

## 🥇 추천 방법 1. Netlify (가장 빠름, 드래그 앤 드롭 방식)
코딩이나 깃허브(GitHub) 연결도 필요 없습니다. 그냥 폴더를 마우스로 끌어다 놓으면 끝납니다.

1. **준비하기**
   - 바탕화면에 임의의 폴더(예: `mbti-test`)를 하나 만듭니다.
   - 그 폴더 안에 지금까지 작업한 `index.html` 파일을 집어넣습니다.
2. **배포 플랫폼 접속**
   - [Netlify Drop (네틀리파이 드롭)](https://app.netlify.com/drop) 사이트에 접속합니다.
3. **가입 및 드래그 앤 드롭**
   - 구글 계정으로 1초 만에 회원가입(Sign up)을 마칩니다.
   - 화면 중앙에 보이는 둥근 원형 공간(`Drag and drop your site output folder here`)에, 방금 바탕화면에 만든 **폴더 자체를 통째로 마우스로 끌어다 놓습니다.**
4. **🎉 배포 완료**
   - 3초 정도 로딩이 끝나면 `https://랜덤영어-숫자조합.netlify.app` 같은 고유 링크가 발급됩니다. 이 주소를 사람들에게 뿌리면 됩니다!
   - *팁: 링크 이름이 안 예쁠 경우, Netlify 설정(Site settings ➔ Domain management)에서 사이트 이름을 `acts29-mbti.netlify.app` 등으로 원하시는 대로 수정할 수도 있습니다.*

---

## 🥈 추천 방법 2. Vercel (빠르고 안정적임)
Netlify와 거의 동일한 방식으로, 요즘 스타트업/개발자들이 가장 사랑하는 트렌디한 방식입니다.

1. **Vercel 접속**
   - [Vercel (버셀)](https://vercel.com/signup) 사이트에 접속하여 구글/깃허브 계정으로 가입합니다.
2. **폴더 업로드 방식을 못 찾겠다면? Vercel CLI 추천 (약간의 터미널 명령어 활용)**
   - Vercel은 원래 깃허브 계정과 연동해서 저장소를 클릭만 하면 배포해주는 데 가장 특화되어 있습니다.
   - 코드를 올린 깃허브 저장소가 있다면, Vercel 대시보드에서 `[Add New...]` ➔ `[Project]` 클릭 후 내 깃허브 레포지토리를 선택하고 `[Deploy]` 버튼만 누르면 30초 만에 끝납니다.
